# Clueminer


Clueminer is a platform for interactive data-mining with special focus on clustering algorithms.


## Building latest development version

Prerequisites:

   * git
   * java (6 or newer)
   * maven

   
      git clone git://github.com/deric/clueminer.git
      cd clueminer
      git submodule init
      git submodule update
      mvn install
